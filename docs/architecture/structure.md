# API

## Сущность

1. Профиль пользователя
   1. Имя (64 символа)
   2. Возраст (16-99)
   3. Гендер (Enum М|Ж)
   4. Интересы (Enum)
   5. Описание (256 символов)

## Функции (Endpoints)

1. Для взаимодействия с анкетой CRUD
    1. CREATE
    2. UPDATE
    3. READ
    4. DELETE
2. Для поиска других анкет
   1. SEARCH
